## REST + PostgreSQL

#### includes

 - JWT auth throw `passport-jwt`
 - login && registration endpoints
 - email verification
 - forgot password
 - user invitation
 - user management (get users, change user role, delete user)
 
 
#### setup

 - copy `.env.example` to `.env` && set up env variables
 - `npm install`
 - `npm run m`
 - `npm run test`
 - `npm run dev`
